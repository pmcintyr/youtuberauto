# YouTube Automation Pipeline

An automated system that monitors YouTube channels, processes new shorts, enhances them with AI-generated metadata using Google's Gemini API (FREE!), and reposts them.

## Features
- 🎯 Monitors specified YouTube channels for new content
- 🎬 Automatically detects and processes YouTube Shorts
- 🤖 AI-powered title and description enhancement using **Google Gemini (FREE)**
- 📊 Automatic thumbnail generation
- 🔄 Scheduled daily execution via GitHub Actions
- 📝 Comprehensive logging and error tracking
- 💰 **Zero cost** - Uses Gemini's free tier!

## Setup

### Prerequisites
- Python 3.10+
- YouTube Data API v3 access
- Google Gemini API key (FREE tier available)
